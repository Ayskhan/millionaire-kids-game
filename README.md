# Детская игра «Кто хочет стать миллионером?»

Офлайн-игра на Python и `pygame-ce` для Windows 10 и Windows 11. Вся игра, интерфейс, вопросы и подсказки полностью на русском языке и подходят для ребёнка 7 лет.

`pygame-ce` совместим с API `pygame`, поэтому код игры остаётся простым, а сборка на современных версиях Python работает стабильнее.`r`n`r`n## Возможности

- главное меню: `Играть`, `Правила`, `Выход`
- 12 уровней с лестницей призов
- 3 подсказки: `50:50`, `Убрать 1`, `Помощь зала`
- минимум 50 встроенных вопросов в отдельном JSON-файле
- дружелюбный крупный интерфейс
- работа без интернета
- необязательные звуки: если файлов нет, игра всё равно запускается

## Требования

- Windows 10 или Windows 11
- Python 3.11+ рекомендуется

## Установка зависимостей

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск игры

```powershell
python main.py
```

## Сборка standalone `.exe` через PyInstaller

```powershell
pyinstaller --noconfirm --windowed --name MillionaireKids --add-data "questions.json;." --add-data "assets;assets" main.py
```

После сборки готовый файл будет лежать в папке `dist\MillionaireKids\`.

## Необязательные звуки

Игра умеет использовать такие файлы:

- `assets\sounds\click.wav`
- `assets\sounds\correct.wav`
- `assets\sounds\wrong.wav`

Если этих файлов нет, ошибок не будет: игра просто запустится без звука.

## Где менять вопросы

Все вопросы лежат в файле `questions.json`. При запуске игра проверяет:

- что вопросов не меньше 50
- что у каждого вопроса ровно 4 варианта ответа
- что у каждого вопроса ровно 1 правильный индекс
- что уровни покрывают всю лестницу призов

Если JSON испорчен, игра покажет понятную ошибку в консоли и не запустится с неверными данными.
